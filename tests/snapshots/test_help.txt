usage: brew-hop-search [-h] [--refresh [DUR]] [--stale [DUR]] [-f] [-c] [-i]
                       [-t] [-L] [-O] [--brew-verify] [-C] [-n N] [--json]
                       [-g] [-q] [-v] [-V] [-H]
                       [query ...]

Fast offline-first Homebrew formula/cask search.

positional arguments:
  query                 Search query (multiple terms matched with AND)

options:
  -h, --help            show this help message and exit
  --refresh [DUR]       Sync refresh (--refresh=DUR: if older than DUR; bare
                        --refresh: force)
  --stale [DUR]         Background refresh if older than DUR (default: 6h)
  -f, --formulae, --formula
                        Search formulae only
  -c, --casks, --cask   Search casks only
  -i, --installed       Search only installed packages
  -t, --taps            Also search formulae/casks from tapped repos
  -L, --local           Search brew's local API cache (offline)
  -O, --outdated        Show outdated packages with upgrade/pin hints
  --brew-verify         Use brew command for outdated (slower, authoritative)
  -C, --cache           Show cache status and exit
  -n N, --limit N       Max results per section (default 20)
  --json                Output raw JSON
  -g, --grep            Greppable output: slug\tversion\turl\n description
  -q, --quiet           Quiet mode: results only, no header/footer
  -v, --verbose         Verbose process output (-v, -vv for more)
  -V, --version         Show version (-VV for commit log and live check)
  -H, --history         Show version history for a package (from install log)
